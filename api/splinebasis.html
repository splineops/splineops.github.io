SplineBasis#

class splineops.bases.splinebasis.SplineBasis(support: int, degree: int, poles: Sequence[float] | None = None)#

Bases: object

Abstract base class for spline basis functions.

Parameters:

  • support (int) – The support of the spline basis, representing the smallest interval in which the basis function is non-zero.

  • degree (int) – The degree of the spline basis function. This parameter is not currently used in the implementation.

  • poles (Optional[Sequence[float]], optional) – Poles of the spline basis function, by default None.

Raises:

  • ValueError – If support or degree is not an integer.

  • TypeError – If poles is not a sequence of real numbers.
